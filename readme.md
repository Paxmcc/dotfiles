# The Dotfiles of my linux setup

if you want to install any of these you can use the install script in the all directory or you can run stow <name-of-dotfile> to install it, but you need the necessary things installed for that, including GNU stow. if you want to install everything run the master-installation.sh to install and symlink everything necessary. After installing anything DO NOT REMOVE THE FOLDER as it will delete the symlink and uninstall the dotfiles.
